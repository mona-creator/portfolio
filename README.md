# <アプリ名>
Business.Succession
## サイト概要
後継者を探している事業主と起業したい人とのマッチングサイト。
### サイトテーマ
事業主と新たな起業家や他の事業主へ橋渡し。

### テーマを選んだ理由
中小零細企業の後継者不足の問題の解決及び新たに事業を取り組みたい人の選択肢を増やす。

### ターゲットユーザ
50歳から60歳で会社を辞めて、次を考えている人。他の事業に取り組みたい事業主。

### 主な利用シーン
再就職やセカンドキャリアを考える時、事業拡大。
